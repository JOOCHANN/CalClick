# 개인정보 처리방침

최종 업데이트: 2026-04-21

CalClick(이하 "서비스")은 한국 개인정보보호법을 준수합니다.

## 1. 수집 항목
- **계정 정보**: 이메일, 비밀번호 (암호화 저장)
- **식사 기록**: 촬영/업로드한 음식 사진, 인식된 음식명, 중량, 칼로리, 기록 시각
- **사용 동의 기록**: 동의 일시

## 2. 수집 및 이용 목적
- 음식 사진 분석 및 칼로리 계산 제공
- 일일/누적 섭취량 기록 및 조회

## 3. 보관 기간
- 계정 삭제 요청 시 **즉시 영구 삭제** (소프트 삭제 미사용)
- 관련 법령에 의해 보존 의무가 있는 경우 해당 기간까지

## 4. 제3자 제공
- 음식 인식을 위해 사진이 **OpenAI (GPT-4o Vision API)** 으로 일시 전송됨 (Cloudflare AI Gateway 경유)
- OpenAI는 API 요청 데이터를 모델 학습에 사용하지 않음 (기본 정책 기준)
- 그 외 제3자 제공 없음

## 5. 삭제권 행사
- 앱 내 **설정 → 계정 삭제**에서 즉시 삭제 가능
- 삭제 시 계정·식사 기록·사진 전부 DB에서 삭제됨

## 6. 안전성 확보 조치
- 전송 구간 HTTPS
- Supabase RLS(Row Level Security)로 타 사용자 데이터 접근 차단
- 비밀번호 해시 저장
